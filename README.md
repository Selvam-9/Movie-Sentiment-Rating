This document defines the requirements for deploying the existing Movie Sentiment Rating Streamlit application to Streamlit Cloud via GitHub.
The application uses a pre-trained Simple RNN model to predict movie review sentiment (positive/negative) and currently runs only locally.
The goal is to make the project deployment-ready, push it to a GitHub repository with proper structure, and deploy it on Streamlit Cloud so it is publicly accessible.

- **App**: The Streamlit application defined in `Sentiment_Score.py` that provides the movie review sentiment analysis interface.
- **Streamlit_Cloud**: The Streamlit Community Cloud hosting platform that deploys Streamlit apps directly from GitHub repositories.
